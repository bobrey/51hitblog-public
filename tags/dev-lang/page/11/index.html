Ssh不自动断开

服务器端设置

vim /etc/ssh/sshd_config
ClientAliveInterval 30
ClientAliveCountMax 3

客户端设置

vim /etc/ssh/ssh_config.    vim ~/.ssh/config
Host *
  ServerAliveInterval 30
  ServerAliveCountMax 3

临时使用一次

ssh -o ServerAliveInterval=30 user@host

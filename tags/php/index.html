BelongsToVShasOne

hasOne

User (id - primary key) => hasOne =>Profile ( user_id - foreing key) [user_id]

BelongsTo

[user_id] Profile (user_id - foreing key) => BelongsTo => User (id - primary key)

小结

If B has an a_id column, then A hasOne or hasMany B

if A has a b_id column, then A belongsTo B.

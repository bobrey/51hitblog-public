NodeJs

1.download [node.exe]

2.download [npm.zip]

http://nodejs.org/dist/npm/

  1. make [node] foler

node

|-nodejs —>node and npm

|-npm-global —>global

|-npm-cache —>cache

  1. set env

move [node.exe] to [node]folder;

move [npmxx.zip] to [node]folder ,and unzip it;

set PATH add xxxxx\node\nodejs

set PATH add xxxxx\node\node-global

then type cmd

#node -v

#npm -v

#npm config set prefix “xxxxx\node\node-global”

#npm config set cache “xxxxx\node\node-cache”

$ npm config delete proxy

|*\*#npm config edit **|

//when proxy

#npm config set registry=http://registry.npmjs.org/

#npm config set strict-ssl false

#npm config set proxy “http://login:pass@host:port/”

#npm config set https-proxy “http://login:pass@host:port/”

5.install express

#npm cache clean

#npm install express -g

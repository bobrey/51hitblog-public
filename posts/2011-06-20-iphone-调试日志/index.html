iphone 调试日志

Mon Jun 20, 2011

大部分人调试程序都是看日志吧，这里我就给大家总结一下iphone程序中添加保存日志的方法。

Objective-C开发程序的时候，有专门的日志操作类NSLog，它将指定的输出到标准的错误输出上（stderr）。我们可以利用它在Xcode的日志输出窗口，或者是输出到具体的文件当中。

下面是我在程序中常用到的日志宏，用DEBUG开关管理，也就是说只有在DEBUG模式下才让日志输出 ：

  <td>
    <pre class="src"><span class="preprocessor"><span>#ifdef</span></span> DEBUG

# define LOG(fmt, …) do { / NSString* file = [[NSString alloc] initWithFormat:@"%s", __FILE__]; / NSLog((@"%@(%d) “ fmt), [file lastPathComponent], __LINE__, ##__VA_ARGS__); / [file release]; / } while(0) # define LOG_METHOD NSLog(@"%s”, __func__) # define LOG_CMETHOD NSLog(@"%@/%@", NSStringFromClass([self class]), NSStringFromSelector(_cmd)) # define COUNT(p) NSLog(@"%s(%d): count = %d/n", __func__, __LINE__, [p retainCount]); # define LOG_TRACE(x) do {printf x; putchar('/n'); fflush(stdout);} while (0) #else # define LOG(…) # define LOG_METHOD # define LOG_CMETHOD # define COUNT(p) # define LOG_TRACE(x) #endif

可以看到，除了标准的用户定义输出外，我还加入了许多有用的信息，比如源程序文件位置，行号，类名，函数名等。具体的应用可以在具体的开发过程中添加、删除。

 

真机测试的时候，可以利用freopen将标准错误输出保存到指定的文件当中，这样就可以在问题发生后分析日志文件。

  <td>
    <pre class="src">- (<span class="type"><strong><span>void</span></strong></span>)<span class="function-name"><span>redirectNSLogToDocumentFolder</span></span>{
<span class="type"><strong><span>NSArray</span></strong></span> *<span class="variable-name"><strong><span>paths</span></strong></span> = NSSearchPathForDirectoriesInDomains(NSDocumentDirectory,NSUserDomainMask, <span class="constant"><span>YES</span></span>);
<span class="type"><strong><span>NSString</span></strong></span> *<span class="variable-name"><strong><span>documentsDirectory</span></strong></span> = [paths objectAtIndex:0];
<span class="type"><strong><span>NSString</span></strong></span> *<span class="variable-name"><strong><span>fileName</span></strong></span> =[<span class="type"><strong><span>NSString</span></strong></span> stringWithFormat:@<span class="string"><span>"%@.log"</span></span>,[<span class="type"><strong><span>NSDate</span></strong></span> date]];
<span class="type"><strong><span>NSString</span></strong></span> *<span class="variable-name"><strong><span>logFilePath</span></strong></span> = [documentsDirectory stringByAppendingPathComponent:fileName];
freopen([logFilePath cStringUsingEncoding:NSASCIIStringEncoding],<span class="string"><span>"a+"</span></span>,stderr);

}

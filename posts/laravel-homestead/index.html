Laravel Homestead

Fri Feb 17, 2017

200 Words|Read in about 1 Min
Tags: php  

Laravel strives to make the entire PHP development experience delightful, including your local development environment. Vagrant provides a simple, elegant way to manage and provision Virtual Machines.

Homestead runs on any Windows, Mac, or Linux system, and includes the Nginx web server, PHP 7.1, MySQL, Postgres, Redis, Memcached, Node, and all of the other goodies you need to develop amazing Laravel applications.

包含的软件

  • Ubuntu 16.04
  • Git
  • PHP 7.1
  • Nginx
  • MySQL
  • MariaDB
  • Sqlite3
  • Postgres
  • Composer
  • Node (With Yarn, PM2, Bower, Grunt, and Gulp)
  • Redis
  • Memcached
  • Beanstalkd

如何使用教程

  1. mkdir xxxproj
  2. cd xxxproj
  3. composer require laravel/homestead --dev
  4. php vendor/bin/homestead make
  5. vagrant up

Remember, you will still need to add an /etc/hosts file entry for homestead.app or the domain of your choice.

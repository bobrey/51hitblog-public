vim 缓冲区操作

查看帮助

#:h buffers

窗口和buffers是不同的

#:vs “foo”

查看buffers

#:ls

切换缓冲

#b 1

tab 切换缓冲区

# .vimrc

#当前行号

set laststatus=2

set statusline=%F:\ %l

#tab切换缓冲区

set hidden “in order to switch between buffers with unsaved change

map :bp

map :bn

tab管理

#tabnew b1

#tabnext

#tabclose

多窗口

#vs xx 垂直

#split xx 水平

窗口之间切换使用

# ctrl +w+w

#ctrl +w + 稍微增加空间

#ctrl +w – 稍微减少空间

#ctrl +w J 增大空间

#ctrl +w H 左侧

#ctrl +w L 右侧

#ctrl +w o 全屏

 

启动 vim server

# vim –servername VIM-SERVER filename

另一个终端上的vim

# vim –servername VIM-SERVER –remote-silent 现在的文件名

git-config 配置文件

Fri Dec 21, 2012

200 Words|Read in about 1 Min
Tags: git  

~/.gitconfig 是最常用的； 对应配置命令要加上–global

git配置文件

~/.gitconfig

[user]

name = fly

email = flyhawk007hm@hotmail.com

[core]

editor = vim

[alias]

ci = commit -a -v

co = checkout

st = status

br = branch

throw = reset –hard HEAD

throwh = reset –hard HEAD^

[color]

ui = true

[push]

default = current

[merge]

tool = meld

/*

  • 用其他用户名在本机提交

*/

# git commit –author = “fly xxx@xx.com” -a -m “fix sth”

查看帮助

# man git config

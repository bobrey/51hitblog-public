初始化 Winsock 调用sock函数创建套接口 函数定义

      SOCKET socket( int af,           // 协议簇 一般 AF_INET 用于 Internet 协议 int type,       // 套接口类型(SOCK_STREAM、SOCK_RGRAM、SOCK_RAM) int protocol // 所用协议 );

返回值 1、没错误返回一个新套接口描述字 返回常量值INVALID_SOCKET

这时可用 WSAGetLastError 函数查系统错误代码。

在winsock2.h中 socket 定义类型为 typedef u_int Socket;

套接口类型

=============================================================

SOCK_STREAM 提供可靠的、双向的、具于连接的字节流传输服务，对应于TCP协议

SOCK_DGRAM 提供不可靠、非连接的数据报服务，通常这种数据报大小固定并且较小，对应于UDP协议

SOCK_RAW 可以读写ICMP、ICMP分组，可以读写内核不处理的包含特殊IP协议段的IP报文，

利用原始套接口加上IP_HDRINCL 选项可以构造自己的IP报头。

创建SOCKET时，TCP、UDP类型 第三个参数 设置为0

(根据 AF_INET+SOCK_STREAM/SOCK_DGRAM)

创建原始套接口时才需要protocol 参数

SOCKET sock = socket(AF_INET, SOCK_STREAM, 0);

SOCKET sock = socket(AF_INET, SOCK_DGRAM, 0);

SOCKET sock = socket(AF_INET, SOCK_RAW, IPPROTO_ICMP);

sap gui 客户端

Thu Jan 12, 2012

300 Words|Read in about 1 Min
Tags:

Windows GUI，最常用和标准的登录方式，能够百分之百的使用 SAP 的所有功能，可以到 ftp://ftp.sap.com/pub/sapgui/win/

JAVA GUI，是针对 JAVA 用户的登录方式，能够使用大部分 SAP 功能，打印功能较弱，能够兼容各种操作系统，比如 Unix、Linux、Solaris、Mac OS 等等，可以到 ftp://ftp.sap.com/pub/sapgui/java 目录下下载，

 

客户端配置

点击new，Description:sap         ConnectionType:R/3

点击Advanced标签，选中Expert Mode,在下面写入conn=/H/服务器域名或者IP地址/S/连接服务器的端口号，这里我填的是：conn=/H/192.168.0.123/S/3200。后面的端口号要注意，每个服务器都是不一样的

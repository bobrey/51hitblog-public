gitbook发布epub

Thu Apr 26, 2018

300 Words|Read in about 1 Min
Tags: git  

  1. 先安装NodeJS和GitBook命令行工具(npm install gitbook-cli -g命令)。 运行 gitbook install，安装GitBook插件。

  2. 构建电子书 运行下面的命令之前，要先使用Summary生成一个’SUMMARY.md'，然后，在书目录下，运行：

  3. $ gitbook build 这将把所有.md文件，转换成.html格式的静态文件，并存放在根目录下的_book目录

  4. Download the Calibre application. After moving the calibre.app to your Applications folder create a symbolic link to the ebook-convert tool: $ln -s ~/Applications/calibre.app/Contents/MacOS/ebook-convert /usr/local/bin

    Generate an ePub file

    $ gitbook epub ./ ./mybook.epub

    Generate a PDF file

    $ gitbook pdf ./ ./mybook.pdf

** 安装Summary 打开命令行；然后，进入书目所在目录，键入如下命令 $ book sm 一个完整的目录文件SUMMARY.md就生成了。 **

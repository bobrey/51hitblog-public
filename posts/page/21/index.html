swiftgen

The Swift code generator for your assets, storyboards, Localizable.strings

install

  • via cocopods If you’re using CocoaPods, you can simply add pod ‘SwiftGen’ to your Podfile.
  • Homebrew 
      brew update
  brew install swiftgen

usage

  • swiftgen images [OPTIONS] DIR
  • swiftgen strings [OPTIONS] FILE
  • swiftgen storyboards [OPTIONS] DIR
  • swiftgen colors [OPTIONS] FILE
  • swiftgen fonts [OPTIONS] DIR

ref:https://github.com/SwiftGen/SwiftGen

but in fact,there is none!

so ,what should we do

just create my.cnf file.

******************

input the socket path

[client]

socket = /usr/local/zend/mysql/tmp/mysql.sock

 

 

******************

maybe it exist

just  open  /usr/local/zend/mysql/data/my.cnf file

//////////////////////////////////////////////////////

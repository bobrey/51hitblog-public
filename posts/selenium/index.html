sudo apt-get install python-pip -y
sudo pip install selenium

Basic Usage

from selenium import webdriver

driver = webdriver.Firefox()
driver.get("http://www.imdb.com/name/nm0331516/")

pagesHTML = driver.page_source
searchBar = driver.find_element_by_id("navbar-query")
searchBar.send_keys("summer glau")
driver.find_element_by_id("navbar-submit-button").click()

firstResult = driver.find_element_by_class_name("result_text")
resultingURL = firstResult.find_element_by_tag_name("a").get_attribute("href")
driver.get(resultingURL)

driver.close()

If you want to use Chrome instead of Firefox, you need to download a chromedriver file from: http://chromedriver.storage.googleapis.com/index.html.

Download the latest version for your distribution, unzip the file, and move the output chromedriver file to /usr/bin. You can do this either through the GUI or you can use bash once you’ve determined the correct file to download.

For chromedriver version 2.10, 32-bit Linux, this looks like:

wget http://chromedriver.storage.googleapis.com/2.10/chromedriver_linux32.zip -P ~/Downloads –quiet unzip ~/Downloads/chromedriver_linux32.zip && sudo mv chromedriver /usr/bin

The only change to the code then would be to replace driver = webdriver.Firefox() with:

driver = webdriver.Chrome()

Other considerations

You may want or need to explicitly set the timeout for Selenium. This is accomplished with:

driver.set_page_load_timeout(30) #Where 30 is the timeout in seconds - an error is thrown if the page has not loaded in 30 seconds

It can also be useful to save a screenshot of the browser’s content. This can be done with:

driver.save_screenshot("imagepath/image.png")

sudo apt-get install tor privoxy -y

echo 'echo "forward-socks5 / localhost:9050 ." >> /etc/privoxy/config' | sudo -s #Append "forward-socks5 / localhost:9050 ." to end of config file

netstat -tulpn | grep 9050
netstat -tulpn | grep 8118

Firefox

#Import webdriver from the selenium library
from selenium import webdriver

#Import the Proxy class from the selenium library
from selenium.webdriver.common.proxy import *

#Build the Proxy object using the locally running Privoxy
port = "8118" #The Privoxy (HTTP) port
myProxy = "127.0.0.1:"+port
proxy = Proxy({
    'proxyType': ProxyType.MANUAL,
    'httpProxy': myProxy,
    'ftpProxy': myProxy,
    'sslProxy': myProxy,
    'noProxy': ''
})

#Launch the Firefox window and visit the given URL
driver = webdriver.Firefox(proxy=proxy)
driver.get("http://www.whatismyip.com/")

#Everything else as normal...

#Close the Firefox window
driver.close()

Chrome

#Import webdriver from the selenium library
from selenium import webdriver

#Build the chrome_options object specifying the locally running Privoxy as the proxy server
chrome_options = webdriver.ChromeOptions()
chrome_options.add_argument('--proxy-server=http://127.0.0.1:8118')

#Launch the Chrome window and visit the given URL
driver = webdriver.Chrome(executable_path="path to chromedriver file you downloaded",chrome_options=chrome_options)
driver.get("http://www.whatismyip.com/")

#Everything else as normal...

#Close the Chrome window
driver.close()

state=driver.find_element_by_class_name("ip-box-left").find_element_by_class_name("the-state").text.encode("utf-8")
country=driver.find_element_by_class_name("ip-box-left").find_element_by_class_name("the-country").text.encode("utf-8")[:-1].strip()
print "State/Region: "+state+"\nCountry: "+country

Selenium Headless(Xvfb)

sudo apt-get install xvfb -y
sudo pip install xvfbwrapper


#Import webdriver from the selenium library
from selenium import webdriver

#Import Xvfb from the xvfbwrapper library
from xvfbwrapper import Xvfb

#Create and start the virtual display
vdisplay = Xvfb()
vdisplay.start()

#Launch the Firefox window and visit the given URL
driver = webdriver.Firefox()
driver.get("https://www.youtube.com/watch?v=cc_szUYJAYE")

#Do stuff!

#Close the Firefox window
driver.close()

#Close the virtual display
vdisplay.stop()

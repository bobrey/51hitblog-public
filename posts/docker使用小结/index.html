Docker使用小结

搭建redis服务器

docker run -p 6379:6379 -v redis_data:/data  --name redis -d redis:latest

搭建mongo服务器

docker run -p 27017:27017 -v mongo_data:/data/db  --name mongo -d mvertes/alpine-mongo

crcom爬虫

编译生成docker镜像

docker build -t crcom:latest .

生成docker容器

docker run -p 6800:6800 --name crcom -d crcom

连接crcom容器，启动爬虫

docker exec -it crcom  /bin/bash


docker exec sleeper echo "hello host from container"
docker exec -d sleeper find / -ctime 7 -name '*log' -exec rm {} \;

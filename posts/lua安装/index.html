Lua安装

安装lua

 sudo apt-get install libreadline-dev
 
 curl -R -O http://www.lua.org/ftp/lua-5.3.4.tar.gz
tar zxf lua-5.3.4.tar.gz
cd lua-5.3.4
sudo make linux test #[这一步有可能会报错，如果有报错下面有解决方法]
sudo make install

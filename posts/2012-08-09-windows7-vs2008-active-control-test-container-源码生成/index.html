windows7 vs2008 Active control test container [源码生成]

Thu Aug 9, 2012

700 Words|Read in about 2 Min
Tags:

在VS 2008中Active X control test container是不以Tools的形式提供，而是以Sample的形式存在。在我的电脑上存在的位置为：C:/Program Files/Microsoft Visual Studio 9.0/Samples/1033。

 

把该目录下的AllVCLanguageSamples.zip拷贝到别的目录（不拷贝也行），解压后找到/AllVCLanguageSamples/C++/MFC/ole/TstCon，双击TstCon.sln就打开工程了。

 

接下来要设置一下，首先是右击Solution（不是项目，有两个项目，一个是TCProps，另一个是TstCon）选择属性，出现一个对话框

。然后配置属性，一定要保证和自己的平台是一样的。默认的是Itanium，我改为Win32。如下图所示：

 

http://p.blog.csdn.net/images/phttp://p.blog.csdn.net/images/p_blog_csdn_net/lanseshenhua/EntryImages/20091002/2009-10-2%2010-03-01.jpg

 

然后分别在两个项目的stdafx.h文件里面添加#define WINVER 0x0501这条语句（我的是Windows XP系统，其他系统请看下表）：

Windows Server 2003 family     _WIN32_WINNT>=0×0502   Windows XP              _WIN32_WINNT>=0×0501   Windows 2000                _WIN32_WINNT>=0×0500   Windows NT 4.0              _WIN32_WINNT>=0×0400   Windows Me              _WIN32_WINDOWS=0×0490   Windows 98              _WIN32_WINDOWS>=0×0410

然后Build Solution，运行程序就可以了.

 

好了，大功告成。

 

 

转自: http://blog.csdn.net/lanseshenhua/archive/2009/10/02/4626897.aspx

 

补充些

1》  Windows7              _WIN32_WINNT>=0×0600

2》 Project -> Properties -> Linker -> Manifest File -> Enable User Account Control（UAC） = False

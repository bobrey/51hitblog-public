免费ssl证书 letsencrypt

Thu May 26, 2016

100 Words|Read in about 1 Min
Tags: letsencrypt   ssl  

git clone https://github.com/letsencrypt/letsencrypt

cd letsencrypt

./letsencrypt-auto –agree-tos –email flyhawk007online@gmail.com certonly –webroot -w /home/www/wwwroot/51hint -d 51hint.com

/etc/letsencrypt/live/51hint.com

<VirtualHost *:443>

SSLEngine on

SSLCertificateFile /etc/letsencrypt/live/51hint.com/cert.pem

SSLCertificateKeyFile /etc/letsencrypt/live/51hint.com/privkey.pem

SSLCertificateChainFile /etc/letsencrypt/live/51hint.com/chain.pem

<Directory home/www/wwwroot/51hint/>

AllowOverride All

Order allow,deny

Allow from all

DocumentRoot home/www/wwwroot/51hint/

ServerName 51hint.com

ErrorLog logs/51hint-error_log

CustomLog logs/51hint-access_log common

sudo crontab -e

0 3 1 * * /usr/local/src/letsencrypt/letsencrypt-auto renew

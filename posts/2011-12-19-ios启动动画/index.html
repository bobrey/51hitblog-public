找到了方案

  • 使用 AVAnimator: http://www.modejong.com/AVAnimator/
  • glgif : https://github.com/jamesu/glgif

用 上面两个载入动画，然后 Root View Controller 使用 UINavigationController ，在它之前创建一个 SplashViewController (这里里面实现动画)， 动画完成后 NavigationController push MainController

参考：http://stackoverflow.com/questions/2474960/how-to-play-animation-movie-instea...

另外一个动画实现方案：http://www.appsamuck.com/day2.html

目前我已经实现，使用的是第二个方案，用 ImageView 动画切换多张图片。

最方便播放视频的方法是使用UIWebView来加载一个NSURL,也即是视屏地址

self.webView =  [[[UIWebView alloc]    initWithFrame:  [[UIScreen mainScreen]    applicationFrame]] autorelease] ;

NSURLRequest   *request = [NSURLRequest requestWithURL:[NSURL URLWithString:string]];

[self.webView loadRequest:request];

Ceph分布式存储系统

Ceph是一个统一的分布式存储系统，设计初衷是提供较好的性能、可靠性和可扩展性。

支持三种接口： Object：有原生的API，而且也兼容Swift和S3的API。 Block：支持精简配置、快照、克隆。 File：Posix接口，支持快照。

特性 ceph ceph
开发语言 c go
数据冗余 副本，纠删码 Reed-Solomon code
动态扩展 HASH 不支持
中心节点 对象存储无中心，cephFS有元数据服务中心点 无中心
存储方式 块、文件、对象 对象存储（分块）
成熟度 高 中
断点续传 兼容S3，分段上传，断点下载 兼容S3，分段上传，断点下载
管理工具 ceph-admin,ceph-mgr,zabbix插件，web管理工具 命令行工具 mc

adobe photoshop等无限试用

删除“/Library/Application Support/Adobe/ ”文件夹下的“SLStore”和“SLCache”文件夹，并清空废纸篓，注意这样做，您之前的所有Adobe软件激活信息都将丢失)

彻底删除以下文件夹。

C:\Program Files (x86)\Common Files\Adobe\SLCache

C:\ProgramData\Adobe\SLStore

　　2. 直接删除 /Library/Application Support/Adobe/SLCache 和 SLStore，

　　3. 直接删除 ~/Library/Application Support/Adobe/OOBE。

　　4. 直接删除 ~/Library/Saved Application State 中全部以“com.adobe”开头的文件夹。

　　5. 进入终端，输入命令“cd /Library/Caches/”然后输出“rm .*”，

　　6. 还是在终端里，输入命令“cd /Users/你的用户名/Library/Preferences/Adobe/”然后输出“rm .*”，一样是删除类似“.E386677211FE”的授权缓存文件。

　　6. 最后重新运行任意一个Adobe的程序，输出新注册的Adobe ID，点击试用。

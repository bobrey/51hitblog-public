Mysql配置

配置环境变量

 MySQL
    C:\MySQL
    PAtH
    添加;%MySQL%\bin;

服务安装

  命令行(管理员)，进入MySQL文件夹的bin文件夹下，
    mysqld -install

my.ini编辑

    [mysql]
    default-character-set=utf8
    [mysqld]
    port=3306
    basedir=C:\MySQL
    datadir=C:\MySQL\data
    max_connections=200
    character-set-server=utf8
    default-storage-engine=INNODB
    sql_mode=NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES

mysql初始化

    mysqld --initialize
    net start mysql

配置账户

    配置mysql root账户
    net stop mysql
    mysqld --skip-grant-tables
    mysql -u root
    flush privileges
    grant all privileges on *.* to 'root'@'localhost' identified by 'root' with grant option;
    flush privileges
    exit
